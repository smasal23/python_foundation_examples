## Overview
This folder contains **Python implementations of basic arithmetic operations**, developed to strengthen foundational programming skills.  
Each script focuses on a single arithmetic concept and is written using clear logic, meaningful variable names, and function-based structure.

These examples form the base for understanding numerical computation in Python, which is essential for automation, data processing, and engineering applications.

---

## Concepts Covered
- Addition of numbers
- Area calculation
- Unit conversion
- Power and cube operations
- Temperature conversion
- Simple interest calculation
- Geometric area computations

---

## File Descriptions

| File Name | Description |
|---------|------------|
| `addition.py` | Adds two numbers and returns a rounded result |
| `area_rectangle.py` | Calculates the area of a rectangle |
| `km_to_m_cm.py` | Converts kilometers to meters and centimeters |
| `equilateral_triangle_area.py` | Computes the area of an equilateral triangle |
| `cube_number.py` | Calculates the cube of a given number |
| `temperature_conversion.py` | Converts Celsius to Fahrenheit |
| `simple_interest.py` | Computes simple interest using standard formula |
