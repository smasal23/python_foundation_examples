# String Operations – Python Foundations

## Overview

This folder contains Python implementations focused on string and digit manipulation, which are essential for handling text-based data and input processing.
Each script demonstrates common string operations using clear logic and Python’s built-in functionality to solve practical problems involving text, encoding, and validation.

These examples help build a solid foundation for text processing, data cleaning, and input validation tasks relevant to automation, analytics, and engineering workflows.

---

## Concepts Covered

* String traversal and manipulation
* Whitespace handling and formatting
* Reversing and rotating strings
* Digit extraction and processing
* Encoding and decoding techniques
* Pattern validation
* ASCII interpretation
* Substring analysis
* Type conversion between strings and numbers

---

## File Descriptions

| File Name                 | Description                                                     |
| ------------------------- | --------------------------------------------------------------- |
| `remove_spaces.py`        | Removes all whitespaces from a string and returns its length    |
| `reverse_string.py`       | Reverses a given string                                         |
| `print_digits.py`         | Prints individual digits of a number                            |
| `sum_of_digits.py`        | Computes the sum of digits of a number                          |
| `odd_even_digits.py`      | Separates even and odd digits from a number                     |
| `run_length_encode.py`    | Compresses strings using run-length encoding (Kata 50)          |
| `reverse_keep_symbols.py` | Reverses characters while preserving symbol positions (Kata 53) |
| `rotate_string.py`        | Rotates characters within a string (Kata 54)                    |
| `decompress_string.py`    | Expands compressed string representations (Kata 63)             |
| `pan_validation.py`       | Validates format of PAN-like strings (Kata 66)                  |
| `ascii_value.py`          | Determines ASCII values of characters (Kata 67)                 |
| `date_format_check.py`    | Validates date string formatting (Kata 69)                      |
| `count_substring.py`      | Counts occurrences of substrings within text (Kata 58)          |
| `reverse_two_words.py`    | Reverses positions of two words in a string (Kata 72)           |

## How to Run

From the root of the repository or the strings folder:

```bash
python filename.py
```

Example:

```bash
python reverse_string.py
```

## Purpose

The objective of this collection is to develop fluency in text manipulation, pattern handling, and data validation techniques in Python.
These skills are foundational for data preprocessing, automation scripting, log parsing, and building robust user-input handling systems.
