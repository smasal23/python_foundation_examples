## Overview
This folder contains **Python implementations demonstrating conditional logic**, one of the core building blocks of programming.  
Each script focuses on decision-making using `if`, `elif`, and `else` statements to solve real-world and logical problems.

These examples help build a strong understanding of control flow, which is essential for automation logic, validations, and rule-based systems.

---

## Concepts Covered
- Decision making using conditional statements
- Comparison and logical operators
- Input validation
- Handling edge cases and errors

---

## File Descriptions

| File Name | Description |
|---------|------------|
| `leap_year.py` | Checks whether a given year is a leap year |
| `odd_even.py` | Determines whether a number is odd, even, or zero |
| `largest_of_three.py` | Finds the largest among three numbers |
| `smallest_of_two.py` | Identifies the smaller of two numbers |
| `days_in_month.py` | Returns the number of days in a given month |
| `number_sign_check.py` | Handles zero, positive, and negative number cases |
