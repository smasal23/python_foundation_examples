## Overview

This folder contains Python implementations demonstrating conditional logic, one of the core building blocks of programming.
Each script applies decision-making using `if`, `elif`, and `else` constructs to solve logical and real-world validation problems.

These examples build a strong understanding of control flow, which is essential for automation logic, rule-based systems, data validation, and engineering workflows.

---

## Concepts Covered

* Decision making using conditional statements
* Comparison and logical operators
* Input validation and branching logic
* Handling edge cases and boundary conditions
* List membership checks
* Divisibility and number property evaluation

---

## File Descriptions

| File Name              | Description                                                    |
| ---------------------- | -------------------------------------------------------------- |
| `leap_year.py`         | Checks whether a given year is a leap year                     |
| `odd_even.py`          | Determines whether a number is odd, even, or zero              |
| `largest_of_three.py`  | Finds the largest among three numbers                          |
| `smallest_of_two.py`   | Identifies the smaller of two numbers                          |
| `days_in_month.py`     | Returns the number of days in a given month                    |
| `number_sign_check.py` | Handles zero, positive, and negative number cases              |
| `between_range.py`     | Checks whether a value lies within a specified range (Kata 33) |
| `composite_check.py`   | Determines whether a number is composite (Kata 34)             |
| `prime_check.py`       | Evaluates whether a number is prime (Kata 61)                  |
| `divisible_by_11.py`   | Checks divisibility of a number by 11 (Kata 71)                |
| `exist_in_list.py`     | Verifies existence of an element within a list (Kata 39)       |
| `count_repetition.py`  | Counts repeated occurrences of elements (Kata 40)              |
